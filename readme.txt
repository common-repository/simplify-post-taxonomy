== Description ==
In many projects and sites we need to built custom posts types and need to create custom taxonomy to implement the business logic properly.
This plugins helps to create custom post types and Taxonomies and relate them without writing a single line of code.
A beautiful Dashbaord Section Interface is provided to do the works.

== Installation ==

**Normal installation**

1. Download the simplify-post-taxonomy.zip file to your computer.
1. Unzip the file.
1. Upload the `simplify-post-taxonomy` directory to your `/wp-content/plugins/` directory.
1. Activate the plugins through the 'Plugins' menu in WordPress.

That's it. You will find a menu "custom posts" in dashboard.
